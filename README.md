# Berlin → Praha 🚴

Велопутешествие Берлин → Прага, август 2026.

**Маршрут:** Radweg Berlin-Dresden + Elberadweg + Labská stezka  
**Дистанция:** ~465 км · **Дней:** 6 · **Набор:** ~800–1 000 м

Berlin → Lübben → Senftenberg → Dresden → Děčín → Litoměřice → Roudnice → Mělník → **Praha**
