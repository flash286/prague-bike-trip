## Цель

Велопутешествие **Берлин → Прага** к концу лета 2026.
- Маршрут: **Radweg Berlin-Dresden (250 км) + Elberadweg/Labská stezka (215 км)** = ~465 км
- Набор высоты: ~800–1 000 м
- Старт: Берлин, август 2026
- Формат: соло, **6 дней / 5 ночёвок**
- Ключевые точки: Berlin → Lübben → Senftenberg → Dresden → Děčín → Litoměřice → Roudnice → Mělník → Praha
- Детальный маршрут: `route-berlin-prague.md`

## Текущая физическая форма

- Недельный объём сейчас: ~30–50 км (апрель 2026 — стартовая точка)
- Типичный выезд: 10–15 км
- **Проблема: боль в спине после 20–30 км** — reach великоват, руки вытянуты, локти не получается подсогнуть → удары от ям идут в локти/плечи/спину
- Колени в порядке
- Пробег за 2025: ~200–300 км суммарно
- Лыжи: катался в марте 2026 (несколько сессий)
- Снова закурил — пульс покоя сейчас ~84 вместо нормальных 62

## Байкфиттинг

### Домашний байкфиттинг (в процессе)

- PWA-гайд: `bikefit.html` — инструкции по замерам, калькулятор посадки, пошаговая настройка
- Диагностика: reach слишком длинный, проставочные кольца уже все наверху, руль выше не поднять
- **Вынос заказан:** PLATT 70 мм / 6° / 31.8 мм / 1-1/8" — 17.99€ Amazon.de (доставка 21.03.2026)
- Установить rise up (логотип сверху) → руль ближе на ~2 см и выше на ~7 мм
- Если 70 мм выноса недостаточно — следующий шаг: сдвинуть седло вперёд по рейлам на 1–1.5 см (проверить KOPS после)

### Профессиональный байкфиттинг (план Б)

Рекомендован **Diagnose Berlin** (⭐ 4.9) — Stargarder Str. 58, 10437 Berlin.
Причина: несколько отзывов именно про боль в пояснице при езде.
Если после 2–3 недель домашней настройки боль не уходит — записаться. Стоимость ~150–200€.

## Обслуживание велосипеда

- Зимовка 2025/2026: тёплый Fahrradraum, без коррозии
- **Смазка цепи:** Muc-Off All-Weather Lube 120 мл — заказан, Amazon.de (15.48€)
- **Давление шин:** покрышки ~40 мм, рекомендация при весе ~90 кг:
  - Город/асфальт: 40–45 PSI (было 50 — жёстко для берлинских дорог)
  - Гравел/грунт: 35–40 PSI
  - Берлин→Прага (микс): 38–42 PSI
  - Минимум: 35 PSI, максимум: 45 PSI
- Тросики: капнуть масло при входе в рубашку
- Подседельный штырь: вытащить, смазать (литиевая смазка), вставить
- Втулки/каретка/рулевая: не трогать при текущем пробеге

## Контекст для разработки

При обновлении PWA:
- Пушить в репозиторий `flash286/prague-bike-trip`
- Файлы: `index.html` (основное), `bikefit.html` (байкфиттинг)
- Пуш из Cowork не работает (нет SSH/gh auth) — пушить вручную с локальной машины
- Стек: vanilla HTML/CSS/JS (no framework), single file
- Шрифты: JetBrains Mono + Unbounded (Google Fonts)
- Цветовая схема: dark, `--bg: #0a0a0f`
- Акцентные цвета по фазам: зелёный → синий → оранжевый → фиолетовый → жёлтый
